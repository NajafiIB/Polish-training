## Core success criteria

- [ ] Can give a stable 2–3 minute self-introduction.
- [ ] Can answer 10 common speaking topics for 60–90 seconds each.
- [ ] Can handle 5 common role-play/dialogue situations.
- [ ] Can describe and compare pictures using one reusable framework.
- [ ] Can write two guided 100-word texts in time.
- [ ] Can complete B1 reading tasks using skimming/scanning.
- [ ] Can complete B1 listening tasks with short answers.
- [ ] Repeated grammar mistakes are reduced enough not to block meaning.

## Red-zone errors

- [ ] Polish spelling: ż, rz, ó, u, ą, ę, ń, ć, ś, ł
- [ ] gender agreement: mój / moja / moje
- [ ] adjective agreement: duży / duża / duże
- [ ] genitive: dużo pracy, nie mam czasu, niedaleko parku
- [ ] locative: w domu, w pracy, w mieście, w mieszkaniu
- [ ] Czech/Slovak interference during speech
