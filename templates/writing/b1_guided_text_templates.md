# B1 Guided Writing Templates

## Purpose

Writing practice must prepare for two guided texts of around 100 words each.

Every task has bullet points. Answer all bullet points. Do not skip any.

## Universal structure

```text
Greeting / opening
Point 1
Point 2
Point 3
Point 4
Closing
```

## Informal email template

```text
Cześć [imię],

Dziękuję za wiadomość. Chcę Ci opowiedzieć o [temat].

Po pierwsze, ...
Po drugie, ...
Poza tym, ...
Na końcu chcę dodać, że ...

Napisz, co o tym myślisz.
Pozdrawiam,
Iman
```

## Blog comment / opinion template

```text
Moim zdaniem [temat] jest bardzo ważny. Uważam, że ...

Po pierwsze, ...
Po drugie, ...
Z mojego doświadczenia wynika, że ...

Podsumowując, myślę, że ...
```

## Complaint / problem message template

```text
Dzień dobry,

Piszę w sprawie [problem]. Niestety ...

Problem polega na tym, że ...
Proszę o ...
Mam nadzieję, że sprawę uda się szybko rozwiązać.

Z poważaniem,
Iman Najafi
```

## Invitation template

```text
Cześć [imię],

Serdecznie zapraszam Cię na [wydarzenie]. Spotkanie odbędzie się [kiedy] w [gdzie].

Planuję zaprosić ...
Będzie ... do jedzenia i picia.
Mam nadzieję, że przyjdziesz.

Do zobaczenia,
Iman
```

## Writing checklist

Before finishing a writing task:

- [ ] Did I answer all bullet points?
- [ ] Is the text around 90–110 words?
- [ ] Did I use greeting/closing if needed?
- [ ] Did I use simple connectors?
- [ ] Did I check endings and Polish letters?
