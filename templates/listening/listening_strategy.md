# B1 Listening Strategy

## Purpose

Listening practice must train answer extraction under time pressure.

## Core strategy

```text
Before listening:
1. Read all questions.
2. Underline keywords.
3. Predict what kind of answer is needed: place, number, person, action, opinion.

First listen:
1. Catch topic and main idea.
2. Write rough notes.
3. Do not panic if you miss something.

Second listen:
1. Confirm exact answer.
2. Fix spelling if possible.
3. Keep answers short.
```

## What to watch for

- numbers,
- dates and times,
- places,
- names,
- negation,
- contrast: ale, jednak,
- reason: bo, ponieważ,
- opinion: uważa, zdaniem, myśli,
- instruction words: zaznacz, wpisz, dopasuj, uzupełnij.

## Short-answer rule

Write only what is needed.

Bad:

```text
On powiedział, że najważniejszym problemem jest zmiana czasu.
```

Better:

```text
zmiana czasu
```
