# B1 Speaking Dialogue Framework

## Purpose

Use this framework for role-play tasks in the speaking exam.

## Universal structure

```text
1. Greeting / opening
2. Ask for details
3. State your need or opinion
4. Explain why
5. Suggest solution
6. Confirm decision / close politely
```

## Useful phrases

### Opening

```text
Dzień dobry, chciałbym zapytać o...
Cześć, mam pytanie.
Chciałbym porozmawiać o...
```

### Asking for details

```text
Gdzie to jest?
Ile to kosztuje?
Kiedy możemy się spotkać?
Jak długo to potrwa?
Czy może Pan/Pani podać więcej informacji?
```

### Giving opinion

```text
Moim zdaniem...
Uważam, że...
Myślę, że to dobry pomysł, ponieważ...
Nie jestem pewien, czy to dobre rozwiązanie.
```

### Suggesting

```text
Proponuję...
Może lepiej...
Możemy też...
Według mnie powinniśmy...
```

### Closing

```text
Dobrze, w takim razie zgadzam się.
Dziękuję za informacje.
Do zobaczenia.
```

## Common role-play contexts

- renting an apartment,
- buying or returning something,
- making or cancelling a reservation,
- asking about a course,
- talking to a doctor,
- arranging a meeting,
- discussing work conditions,
- complaining politely,
- planning a trip with a friend.
