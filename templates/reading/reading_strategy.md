# B1 Reading Strategy

## Purpose

Reading practice is not slow translation. It is exam answer extraction.

## Core strategy

```text
1. Skim the full text quickly.
2. Identify topic, people, places, dates, numbers.
3. Read the question.
4. Find the matching fragment in the text.
5. Answer only from the text.
6. Do not use general knowledge.
```

## What to underline

- names,
- dates,
- places,
- numbers,
- contrast words: ale, jednak, chociaż,
- negation: nie, nigdy, żaden,
- cause/effect: bo, ponieważ, dlatego,
- opinion words: uważa, myśli, zdaniem.

## Short-answer rule

Use short answers. Do not write full sentences unless needed.

Example:

```text
Question: Gdzie mieszka bohater?
Answer: w Krakowie
```

## Common traps

- answer sounds true but is not in the text,
- question uses synonym, not the exact word,
- one answer has too much information,
- negation changes meaning,
- similar numbers/dates are distractors.

## Drill format

For every reading task, record:

```text
Text type:
Task type:
Score:
Wrong answers:
Why wrong:
Words/phrases to reuse:
```
